### MVC思想
业务逻辑，数据，界面分离
* M 模型---数据库
* V 视图---界面
* C 控制器---从视图中读取数据
### 数据库关联
用户管理---客户（可为空），权限
权限管理---权限
客户管理---最基础的信息
